# cachyOS

config fastfetch+ghostty on cachyOS (Arch)

Ghostty [https://github.com/ghostty-org/ghostty](https://ghostty.org/docs/install/binary)

## Install

`pacman -S ghostty`

## Creat config file

`mkdir -p ~/.config/ghostty`

`touch ~/.config/ghostty/config`

## Just tiny config for me 

```
# Background color
# Options: Color: #RRGGBB
background = #241f31

# Window width in cells
# Options: Number
window-width = 120

# Window height in cells
# Options: Number
window-height = 40

```

## Fastfetch 

`mkdir -p ~/.config/fastfetch`

`touch ~/.config/fastfetch/config`
